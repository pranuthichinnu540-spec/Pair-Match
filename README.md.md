# 🎮 Pair Match - Multi-Season Edition

A fun and interactive Memory/Matching Card game with **multiple seasons** and different emoji collections built with HTML, CSS, and JavaScript!

## 🎯 Features

✨ **Game Features:**
- 🌍 **5 Unique Seasons** with different collections
  - Season 1: Emojis 😊
  - Season 2: Animals 🐸
  - Season 3: Places 🏰
  - Season 4: Fruits 🍎
  - Season 5: Vegetables 🥕
- 🎨 Beautiful gradient UI with smooth animations
- 🎯 Match pairs of identical items
- 📊 Track moves, matches, and time
- ⭐ Difficulty levels (Easy, Medium, Hard)
- 🏆 Star rating based on performance
- 📱 Fully responsive design
- 🎵 Smooth card flip animations

## 🎮 How to Play

1. **Choose a Season** - Click on any season button
2. **Click cards** to flip them
3. **Match pairs** by clicking two cards with the same item
4. **Complete the game** by matching all pairs
